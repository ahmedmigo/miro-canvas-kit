# Miro Canvas kit (Formates)

This is a code bundle for Miro Canvas kit (Formates). The original project is available at https://www.figma.com/design/uXEU5OcpqnJrWzzgal1fe8/Miro-Canvas-kit--Formates-.

## Running the code

Run `bun install` to install the dependencies.

Run `bun run dev` to start the development server.
